Unordered list

If you use a list, include more than one list item. If you only have one list item, remove the list and rephrase the information using one or two sentences. Doing so removes unnecessary structure. Use only one type of list at a time; avoid * foo: A tool that ..., where * signifies an unordered list and : signifies a definition list.

Definition list

Use an definition (or variable) list for complementary information that includes a term and a definition.

Ordered list

Use an ordered (or numbered) list for information that needs to be performed in a particular order. If only part of the list needs to be performed in a particular order, write a default order that you want your reader to use.

Unordered list

Use an unordered (or bulleted) list for information that does not need to be performed in a particular order.
